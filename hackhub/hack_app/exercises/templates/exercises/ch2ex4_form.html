Preparation

After configuring the proxy server, in your browser, navigate to Metasploitable's IP Address. From there, you can access 4 intentionally vulnerable web application. In Burp Suite, navigate to proxy > intercept. Here you are able to enable or disable the intercepter. When enabled, the intercepter will display the HTTP requests and allow you to view or modify them before sending them. Therefore, if you have already noticed that web pages are not loading properly when it is enabled, it is because you need to manually forward the requests using the "Forward" button.

Visit the DVWA web application using the link provided. Turn on the interceptor and refresh the page. Using the intercepted request answer Question 1.
