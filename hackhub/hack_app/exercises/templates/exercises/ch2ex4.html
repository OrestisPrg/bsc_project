Essential Tools

2.4: Burp Suite

Take Quiz

Burp Suite is a graphical Web Penetration Testing framework specialised in vulnerability detection. With a collection of tools available, it is able identify and verify attack vectors web applications are vulnerable to and provide feedback on them to the user.

Fundamentally, Burp Suite can intercept and analyse traffic when the user browses a web application by configuring a web browser to route traffic through Burp Suite's proxy server. The proxy, acting as man-in-the-middle, can capture each request to and from the target web application which can then be manipulated or replayed revealing potential injection points and unintended usages of the application that can be abused.

Kali Linux comes with the free community version of Burp Suite pre-installed, which includes a number of different tools that can be useful to a penetration testing attack.

Tools

  • HTTP Proxy: Captures traffic from/to web application by operating as a proxy server, from which the requests pass through.
  • Scanner: Performs secuirity scans, detecting potential vulnerabilities of a web application automatically.
  • Intruder: Performs automated attacks on web applications. By generating malicious HTTP requests it can attempt numerous attacks such as SQL Injections, Cross Site Scripting (XSS) as well as brute-force attacks.
  • Spider: Browses the web application systematically in order to discover its overall structure, content and functionality and create a map of its internal architecture.
  • Repeater: Manipulates, manages and repeats HTTP requests in order to test the target's security and detect vulnerabilities.
  • Decoder
  • Comparer
  • Extender
  • Sequencer

Configuration

Before intercepting traffic, you need to configure your web browser to route traffic through Burp's proxy. To do that on your Kali VM:

  1. Launch Burp Suite.
  2. From the tab menu, select the proxy tab and then the options sub-tab.
  3. Click on the cog button on the left and select "Restore Defaults".
  4. You should see an entry in the table of this panel with interface set to "127.0.0.1:8080". This is the default port the proxy is configured to.
  5. Launch Firefox from the taskbar on the left.
  6. From the menu on the top right, select "Preferences".
  7. On the "General" tab scroll down until you see the "Network Proxy" section and click the "Settings..." button.
  8. Select the "manual proxy configuration" option and in the "HTTP Proxy" field enter "127.0.0.1".
  9. Check the "Use this proxy server for all protocols" box.
  10. Click the OK button and close any remaining dialogs.

If you followed the above steps correctly the network traffic can now be intercepted by Burp Suite.
